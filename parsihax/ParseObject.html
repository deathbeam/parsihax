abstract ParseObject<T>(Vector<ParseFunction<T>>)

package parsihax

The ParseObject object is a wrapper for a parser function. Externally, you use one to parse a string by calling var result = SomeParseObject.apply('Me Me Me! Parse Me!');

Variables

@:implapply:ParseFunction<T>

Getting ParseObject.apply from a parser (or explicitly casting it to ParseFunction returns parsing function String -> ?Int -> ParseResult<A> (or just ParseFunction), that parses the string and returns ParseResult<A>.

Changing ParseObject.apply value changes parser behaviour, but still keeps it's reference, what is really usefull in recursive parsers.
